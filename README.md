
# Next Js Starter

Next js Starter is created in

Next.Js v-13

TypeScript

Yarn

Tailwind css

Eslint





## Installation

Clone Project

```bash
 git clone https://github.com/Anishali2/my-starter
```
Install packages

```bash
 yarn
```  
Run project

```bash
 yarn dev
```
